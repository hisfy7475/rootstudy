# 핵심 요구사항 정리

## 프로젝트 개요

- **프로젝트명**: 독서실 학습 관리 시스템
- **프로젝트 유형**: 웹 애플리케이션 (반응형) + 관리자 웹사이트
- **목적**: 독서실 학생들의 실시간 학습시간 관리, 몰입도 측정, 등원 목표 관리 및 학부모-관리자 간 소통 시스템 구축
- **주요 사용자**: 학생, 학부모, 관리자(실장)

## 핵심 기능

### 학생 시스템

1. **실시간 학습시간 관리**
   - CAPS 시스템(지문 인식) 연동으로 실제 입실 시간만 학습시간으로 자동 계산
   - 메인 화면에 오늘의 학습시간, 주간 목표 시간, 달성률 표시
   - 실시간으로 학습시간 증가하는 형태로 구현

2. **등원 예정시간 목표 관리**
   - 학생이 본인의 등원 예정시간을 미리 설정
   - 목표 시간보다 늦게 오면 자동으로 벌점 부여
   - 목표 달성 시 상점 부여

3. **과목 설정 기능**
   - 학생이 현재 공부하고 있는 과목을 직접 설정
   - 관리자 화면에서 학생의 현재 학습 과목 표시

4. **몰입도 확인**
   - 관리자가 입력한 몰입도 점수 확인 가능
   - 주간 단위 몰입도 리포트 확인

5. **3자 채팅**
   - 학생-학부모-관리자 3자 간 채팅 기능
   - 와이파이 사용 요청 등 소통

### 학부모 시스템

1. **학생 상태 확인**
   - 학생의 입실/퇴실/외출 상태 확인
   - 현재 학습 과목 확인
   - 몰입도 확인

2. **스케줄 승인**
   - 학생이 등록한 스케줄에 대한 승인 기능

3. **3자 채팅 참여**
   - 학생-관리자와 함께 채팅 참여

4. **회원가입**
   - 학생 가입 후 생성되는 코드로 학부모 등록

### 관리자 시스템

1. **학생 현황 대시보드**
   - 전체 학생 입실/퇴실/외출 상태 확인
   - 학생별 현재 학습 과목 표시
   - 과목 미설정 학생 직접 입력 가능

2. **몰입도 관리**
   - 실장들이 웹에서 직접 몰입도 점수 입력
   - 완전 몰입 10점, 인강 시청 8점, 졸고 있음 5점 등
   - 몰입도 리포트 출력 및 정렬 기능

3. **상벌점 관리**
   - 자동 벌점 부여 (지각, 등원 목표 미달성)
   - 자동 상점 부여 (등원 목표 달성)
   - 수동 상벌점 부여 기능

4. **알림 관리**
   - 지각, 결석 시 학부모에게 카카오 알림톡 자동 발송

5. **데이터 관리**
   - 모든 데이터 엑셀로 다운로드 가능

6. **3자 채팅 관리**
   - 학생-학부모 채팅방 참여 및 관리

## 기술적 요구사항

- **플랫폼**: iOS, 안드로이드, 아이패드 모두 지원 (반응형 웹)
- **외부 연동**: CAPS(캡스) 시스템 API 연동 필수 (입실/퇴실 지문 데이터)
- **네트워크**: 독서실 IP 허용 시스템에 웹사이트 IP 추가 필요 (하이온넷 요청)

## 개발 방식

- **웹사이트 우선 개발**: 앱보다 웹사이트로 먼저 개발 (빠르고 비용 효율적)
- **추후 앱 통합 가능**: 웹으로 먼저 개발 후, 추후 기존 앱과 통합하는 방안 검토

## 참고사항

- 기존 앱에는 단어 테스트, 성적 관리, 멘토링 클리닉 신청, 지각/결석/조퇴 신청, 주간 공지사항 등의 기능이 있으나, 이번 프로젝트는 **학습 관리 기능에 집중**
- 기존 시스템은 최초 입실과 마지막 퇴실 시간만 계산하여 허수가 많았음 → 실제 입실 시간만 계산하는 것이 핵심
