# 🗂️ 주요 개발 범위

## 1. 학생 웹 (반응형 웹사이트, 앱 아님)

- 회원가입/로그인 (이메일, 소셜 로그인)
- 메인 대시보드 (학습시간, 목표 달성률, 입실 상태 표시)
- 실시간 학습시간 표시 (CAPS 연동)
- 등원 예정시간 목표 설정 기능
- 현재 학습 과목 설정 기능
- 몰입도 점수 확인 (관리자 입력)
- 주간 몰입도 리포트 확인
- 상벌점 내역 확인
- 3자 채팅 (학부모-관리자 포함)

## 2. 학부모 웹 (반응형 웹사이트, 앱 아님)

- 회원가입/로그인 (학생 코드로 연결)
- 학생 현재 상태 확인 (입실/퇴실/외출)
- 학생 학습 과목 확인
- 학생 몰입도 확인
- 학생 스케줄 승인 기능
- 3자 채팅 참여

## 3. 관리자 웹사이트

- 관리자 로그인
- 학생 현황 대시보드 (입실/퇴실/외출 상태, 현재 학습 과목)
- 학생 과목 직접 입력 (미설정 시)
- 몰입도 점수 입력 기능
- 몰입도 리포트 조회/출력/정렬
- 상벌점 관리 (자동/수동)
- 학부모 알림 관리 (지각/결석 시 카카오 알림톡 발송)
- 학생/학부모 목록 관리
- 전체 데이터 엑셀 다운로드
- 3자 채팅 관리

## 4. 시스템 통합 및 배포

- CAPS API 연동 (입실/퇴실 지문 데이터)
- 카카오 알림톡 연동 (학부모 알림용)
- 서버 구축 및 배포
- 도메인 연결 및 SSL 설정
- IP 허용 요청 가이드 (하이온넷)
