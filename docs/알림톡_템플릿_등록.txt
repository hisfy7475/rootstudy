================================================================================
네이버 클라우드 SENS 알림톡 템플릿 등록 가이드
================================================================================

■ 템플릿 등록 위치
  네이버 클라우드 콘솔 > SENS > Biz Message > 알림톡 템플릿 > 템플릿 등록

================================================================================


【템플릿 1】 공지사항 알림
--------------------------------------------------------------------------------

템플릿 이름: 루트스터디센터_공지사항

템플릿 내용:
--------------------------------------------------------------------------------
[루트 스터디센터 공지]

#{내용}

- 루트 스터디센터
--------------------------------------------------------------------------------

카테고리: 기타
메시지 유형: 기본형 (텍스트)
강조 표기: 사용 안 함


================================================================================


【템플릿 2】 중요 공지사항 알림 (제목 포함)
--------------------------------------------------------------------------------

템플릿 이름: 루트스터디센터_중요공지

템플릿 내용:
--------------------------------------------------------------------------------
[루트 스터디센터]

#{제목}

#{내용}

- 루트 스터디센터
--------------------------------------------------------------------------------

카테고리: 기타
메시지 유형: 강조 표기형
강조 표기 제목: #{제목}


================================================================================


【템플릿 3】 학부모 알림 (학생명 포함)
--------------------------------------------------------------------------------

템플릿 이름: 루트스터디센터_학부모알림

템플릿 내용:
--------------------------------------------------------------------------------
[루트 스터디센터]

#{학생명} 학부모님께 안내드립니다.

#{내용}

- 루트 스터디센터
--------------------------------------------------------------------------------

카테고리: 기타
메시지 유형: 기본형 (텍스트)


================================================================================


■ 등록 시 주의사항

1. 변수는 #{변수명} 형식으로 작성
2. 템플릿 내용은 검수 승인 후 수정 불가 (새로 등록해야 함)
3. 검수 기간: 영업일 기준 1~3일 소요
4. 승인 후 템플릿 코드를 환경변수에 입력:
   NCLOUD_KAKAO_TEMPLATE_CODE=승인된템플릿코드

■ 추천 템플릿

  → 【템플릿 1】 공지사항 알림 을 먼저 등록하세요.
    가장 범용적으로 사용 가능합니다.
