## 📌 프로젝트 개요

루트스터디에서 현재 반응형 웹으로 운영 중인 학생·학부모 시스템을 iOS/Android 네이티브 앱으로 전환하고, 신규 멘토링/클리닉 신청 기능을 함께 개발하는 통합 프로젝트입니다. React Native(Expo) 기반의 앱 쉘을 통해 푸시 알림·자동 로그인·파일 첨부 등 네이티브 기능을 제공하며, 앱스토어 정식 배포까지 포함합니다. 아울러 급식 신청·결제, 멘토링/클리닉 신청, 기존 미완료 개선사항을 한 번에 반영하여 학습 관리 플랫폼으로서의 완성도를 높입니다.

**핵심 특징:**

- iOS/Android 네이티브 앱 (React Native Expo 웹뷰 방식, 빠른 전환 + 낮은 유지 비용)
- 학생·학부모 대상 네이티브 푸시 알림 (공지·채팅·멘토링 결과 실시간 수신)
- 급식 신청·결제 (나이스페이먼츠 연동, 중식/석식 센터별 관리)
- 멘토링/클리닉 신청 (학생·학부모 직접 신청, 관리자 스케줄 등록·확정·알림)
- 자동 로그인 + 파일·사진 첨부 (채팅·공지사항)
- 기존 미완료 개선 항목 일괄 반영 (몰입도·상벌점·스케줄·채팅·회원 등)

## 🗂️ 주요 개발 범위

### 1. React Native 앱 쉘 (학생·학부모 공통)

- Expo 기반 iOS/Android 앱 프로젝트 초기 구성
- 기존 반응형 웹 임베드 (WebView 연동)
- 앱 아이콘 / 스플래시 스크린 설정
- 딥링크 처리 (앱 내 화면 이동)
- App Store / Google Play 빌드 및 배포

### 2. 네이티브 푸시 알림 시스템

- Expo Push Notification (FCM/APNs) 설정 및 토큰 관리
- 공지사항 신규 등록 시 자동 푸시 발송
- 채팅 메시지 수신 시 푸시 알림
- 멘토링 신청·확정·거절 결과 푸시 알림
- 서버 측 푸시 발송 API 구축
- 알림 수신 후 앱 내 해당 화면 딥링크 이동

### 3. 자동 로그인 / 세션 관리

- Expo SecureStore 기반 토큰 영구 저장
- 앱 재실행 시 자동 로그인 처리
- 비밀번호 재설정 (6자리 코드 방식)

### 4. 파일·사진 첨부 기능

- 채팅방: 이미지·파일 전송 (카메라/갤러리 접근)
- 공지사항: 관리자 파일·이미지 첨부 업로드
- 파일 미리보기 / 다운로드 처리

### 5. 급식 신청·결제 시스템 (학생·학부모)

- 급식 메뉴 목록 조회 (센터별 중식/석식)
- 급식 신청 및 결제 (나이스페이먼츠 연동)
- 신청 취소 및 환불 (식사일 2일 전까지)
- 신청 내역 조회

### 6. 급식 관리 (관리자 웹)

- 급식 상품 등록 (센터, 식사유형, 가격, 판매기간)
- 메뉴 입력 및 수정
- 신청 현황 조회 및 관리
- 관리자 상시 취소·환불 처리

### 7. 멘토링/클리닉 신청 시스템 (학생·학부모)

- 멘토링 캘린더 (신청 가능 슬롯 날짜별 표시)
- 슬롯 선택 (시간대·멘토·유형·잔여 좌석)
- 신청자 정보 입력 및 신청 완료
- 신청 내역 조회 (신청중/확정/취소 상태별)
- 신청 취소 (취소 가능 시간 전)

### 8. 멘토링 스케줄 관리 (관리자 웹)

- 멘토(선생님) 등록·관리
- 슬롯 등록 (멘토·날짜·시간대·과목·유형·정원)
- 주간 캘린더 뷰 (슬롯별 신청 현황 확인)
- 신청 확정·거절 처리 및 알림톡·푸시 발송
- 신청 내역 목록 (날짜별·학생별 필터)
- 관리자 강제 취소

### 9. 기존 웹 기능 개선 (미완료 항목 일괄)

- **몰입도**: 지난 시간 추가 입력 / 핸드폰 제출 항목 / 학생 리포트 노출 / 과목별 학습시간 띠그래프
- **상벌점**: 30점 강제퇴원 알림·상계 / 분기별 초기화 / 100점 상품권 초기화 / 학부모 메뉴 상벌점 / 관리자 UX 개선
- **스케줄**: 일회성 자동 삭제 / 정기스케줄 기간 표기 / 관리자 화면 개선 / 결석 기능 추가
- **채팅**: 출석부 채팅방 연결 / 메시지 삭제 / 자주 쓰는 멘트
- **회원**: 미승인 입반 전환 / 탈퇴 오류 수정 / 지점 변경 처리 / 학부모 정렬
- **오류 수정**: 과목 오입력 리셋 / 목표 학습시간 미반영

## 🧑‍💻 기술 스택

### App (신규)
- **React Native (Expo)**: iOS/Android 네이티브 앱 쉘
- **Expo WebView**: 기존 웹 임베드
- **Expo SecureStore**: 토큰 보안 저장
- **Expo Notifications**: FCM/APNs 푸시 알림

### Frontend (기존 유지)
- **Next.js + TypeScript**: 웹 인터페이스
- **Tailwind CSS**: 반응형 디자인

### Backend & Database (기존 유지)
- **Supabase**: PostgreSQL 데이터베이스, 인증 시스템, 실시간 기능, Storage

### External Integration (추가)
- **나이스페이먼츠**: 급식 신청 결제·환불
- **FCM / Apple APNs**: 푸시 알림 인프라
- **NCloud SENS (카카오 알림톡)**: 기존 연동 + 멘토링 알림 템플릿 추가

### 배포 (기존 유지 + 앱 추가)
- **Vercel**: 웹 프론트엔드
- **App Store Connect**: iOS 앱 배포
- **Google Play Console**: Android 앱 배포
