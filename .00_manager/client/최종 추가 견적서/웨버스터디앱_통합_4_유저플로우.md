## 전체 시스템 구조

```mermaid
graph TD
    A[앱 실행] --> B{자동 로그인?}
    B -->|성공| C[메인 웹뷰]
    B -->|실패| D[로그인 화면]
    D --> C

    C --> E[공지 푸시 알림]
    C --> F[채팅 푸시 알림]
    C --> G[급식 신청]
    C --> H[멘토링 신청]
    C --> I[기존 기능들]

    E --> C
    F --> C
```

## 1. 앱 쉘 / 자동 로그인 플로우

```mermaid
flowchart TD
    A([앱 실행]) --> B[Expo 초기화]
    B --> C{SecureStore\n토큰 존재?}
    C -->|없음| D[로그인 화면]
    C -->|있음| E[Supabase 세션 복원]
    E --> F{토큰 유효?}
    F -->|유효| G[WebView에 쿠키 주입]
    F -->|만료| H[토큰 갱신 시도]
    H --> I{갱신 성공?}
    I -->|성공| G
    I -->|실패| D
    G --> J[메인 WebView 화면]
    D --> K[이메일/비밀번호 로그인]
    K --> L{로그인 성공?}
    L -->|성공| M[토큰 SecureStore 저장]
    M --> G
    L -->|실패| D
```

## 2. 비밀번호 재설정 플로우

```mermaid
flowchart TD
    A([비밀번호 재설정 요청]) --> B[이메일 입력 화면]
    B --> C[이메일로 6자리 코드 발송]
    C --> D[코드 입력 화면]
    D --> E{코드 유효?}
    E -->|유효| F[새 비밀번호 입력 화면]
    E -->|만료/오류| G[재시도 안내]
    F --> H[비밀번호 변경 완료]
    H --> I[로그인 화면으로 이동]
```

## 3. 푸시 알림 시스템 플로우

```mermaid
flowchart TD
    A([앱 최초 설치]) --> B[알림 권한 요청\niOS 명시적 모달]
    B --> C{허용?}
    C -->|허용| D[Expo 푸시 토큰 발급]
    C -->|거부| E[알림 미수신\n앱 정상 사용]
    D --> F[Supabase에 토큰 저장]

    G([공지 등록 / 채팅 전송 / 멘토링 이벤트]) --> H[서버 발송 API 호출]
    H --> I[대상 토큰 조회]
    I --> J[Expo Push API 호출]
    J --> K{포그라운드?}
    K -->|포그라운드| L[인앱 알림 배너 표시]
    K -->|백그라운드/종료| M[시스템 푸시 알림]
    M --> N{알림 탭}
    N --> O[딥링크 → 해당 화면 이동]
```

## 4. 파일·사진 첨부 플로우

```mermaid
flowchart TD
    A([채팅 첨부 버튼 탭]) --> B[선택 모달]
    B --> C{선택}
    C -->|카메라| D[카메라 권한 요청]
    C -->|갤러리| E[갤러리 권한 요청]
    C -->|파일| F[파일 피커]
    D --> G[사진 촬영]
    E --> H[이미지 선택]
    F --> I[파일 선택]
    G & H --> J[이미지 압축 처리]
    J & I --> K[미리보기 확인]
    K --> L[Supabase Storage 업로드]
    L --> M{업로드 성공?}
    M -->|성공| N[채팅 메시지로 URL 저장]
    M -->|실패| O[오류 안내 토스트]
    N --> P[채팅 화면에 썸네일 표시]
    P --> Q{썸네일 탭}
    Q --> R[이미지 풀스크린 뷰어]
```

## 5. 급식 신청·결제 플로우

```mermaid
flowchart TD
    A([급식 탭 진입]) --> B[급식 목록 화면]
    B --> C[날짜/유형 탭 선택]
    C --> D[급식 상품 카드 목록]
    D --> E{신청 상태?}
    E -->|신청가능| F[급식 상세 화면]
    E -->|신청완료| G[신청 상세 조회]
    E -->|마감/기간외| H[비활성 표시]
    F --> I[신청 버튼]
    I --> J[신청 확인 화면\n자동 입력 정보 확인]
    J --> K[결제 버튼]
    K --> L[나이스페이먼츠 결제 WebView]
    L --> M{결제 결과}
    M -->|성공| N[신청 DB 저장]
    N --> O[확인 알림톡 발송]
    O --> P[결제 완료 화면]
    M -->|실패/취소| Q[결제 실패 화면]
    Q --> F

    G --> R{취소 가능 기한?}
    R -->|가능| S[취소 확인 모달]
    R -->|불가| T[취소 불가 안내]
    S --> U[환불 API 호출]
    U --> V[신청 상태 취소 업데이트]
    V --> W[취소 확인 알림톡]
```

## 6. 급식 관리 플로우 (관리자)

```mermaid
flowchart TD

    E -->|중복 없음| F[저장 완료]
    E -->|중복 있음| G[오류 안내]
    F --> H[메뉴 입력 화면]
    H --> I[메뉴 텍스트 저장]

    B --> J[신청 현황 조회]
    J --> K[신청자 목록 화면]
    K --> L{강제 취소}
    L --> M[환불 API 호출]
    M --> N[취소 알림톡 발송]
```

## 7. 멘토링 신청 플로우 (학생·학부모)

```mermaid
flowchart TD
    A([멘토링 탭 진입]) --> B[멘토링 캘린더 화면]
    B --> C{날짜 선택}
    C -->|슬롯 없음| D[해당 날짜 신청 불가]
    C -->|슬롯 있음| E[슬롯 목록 화면]
    E --> F{슬롯 선택}
    F -->|잔여 좌석 없음| G[만석 표시 - 선택 불가]
    F -->|잔여 있음| H[신청자 정보 확인 화면]
    H --> I[신청 완료 버튼]
    I --> J{중복 신청?}
    J -->|중복| K[중복 신청 안내]
    J -->|정상| L[신청 DB 저장 - 상태: 신청중]
    L --> M[관리자 신청 접수 알림 발송]
    M --> N[학생 신청 접수 알림톡]
    N --> O[신청 완료 화면]

    P([신청 내역 탭]) --> Q[신청 목록 화면]
    Q --> R[신청 상세 화면]
    R --> S{상태?}
    S -->|신청중/확정| T{취소 가능 시간?}
    S -->|취소 완료| U[조회만 가능]
    T -->|가능| V[취소 확인 모달]
    T -->|불가| W[취소 불가 안내]
    V --> X[취소 처리 + 좌석 복구]
    X --> Y[관리자 취소 알림]
```

## 8. 멘토링 스케줄 관리 플로우 (관리자)

```mermaid
flowchart TD
    A([관리자 멘토링 메뉴]) --> B{선택}
    B -->|멘토 관리| C[멘토 목록 화면]
    C --> D[멘토 등록/수정 화면]
    D --> E[저장 완료]

    B -->|슬롯 관리| F[슬롯 목록 화면]
    F --> G[슬롯 등록 화면]
    G --> H{중복 슬롯 체크}
    H -->|중복| I[오류 안내]
    H -->|정상| J[슬롯 저장]

    B -->|주간 캘린더| K[주간 캘린더 뷰]
    K --> L[슬롯 블록 클릭]
    L --> M[신청자 목록 모달]

    B -->|신청 내역| N[신청 목록 화면\n날짜·학생·상태 필터]
    N --> O[신청 상세 모달]
    O --> P{처리 선택}
    P -->|확정| Q[확정 처리]
    Q --> R[확정 알림톡 + 푸시 발송]
    P -->|거절| S[거절 처리]
    S --> T[거절 알림톡 + 푸시 발송]
    P -->|강제취소| U[강제 취소 처리]
    U --> V[취소 알림 발송]
```

## 9. 기존 기능 개선 플로우

```mermaid
flowchart TD
    subgraph 몰입도
    A1[학생 몰입도 입력] --> A2[핸드폰 제출 체크박스 추가]
    A3[지난 시간 추가 입력] --> A4[수동 학습시간 저장]
    A5[학생 리포트 조회] --> A6[과목별 띠그래프 표시]
    end

    subgraph 상벌점
    B1[상벌점 적립/차감] --> B2{임계값 도달?}
    B2 -->|30점| B3[강제퇴원 알림 발송]
    B2 -->|100점| B4[상품권 지급 + 초기화]
    B5[분기 초기화 배치] --> B6[일괄 초기화]
    end

    subgraph 채팅
    C1[출석부 학생 행 클릭] --> C2[채팅방 바로가기 버튼]
    C3[메시지 길게 누르기] --> C4[삭제 옵션]
    C5[자주 쓰는 멘트 버튼] --> C6[단축 문구 삽입]
    end
```
