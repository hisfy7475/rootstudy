# Step 1: 요구사항 정리 — 웨버스터디앱 통합 플랜

> 기존 8주 플랜(앱 쉘·푸시·자동로그인·파일첨부·급식신청·기존개선)에
> 클라이언트 추가 요청 기능(멘토링/클리닉 신청)을 통합한 하나의 견적입니다.
> 모의고사 신청 및 AI 학습통계 리포트는 클라이언트 의사 확인 후 제외합니다.

---

## 프로젝트 배경

- 현재 웨버스터디는 **Next.js 반응형 웹**으로 학생·학부모 시스템 운영 중
- iOS/Android **네이티브 앱** 으로 전환하여 앱스토어 정식 배포 필요
- 기존에 제공하던 독서실 관리 시스템(출석, 채팅, 스케줄, 몰입도 등) 위에 신규 기능 추가
- 클라이언트가 과거에 사용하던 앱(LUsoft Android/iOS)은 **레거시 분석 참고용만** 사용, 코드 재사용 없음

---

## 사용자 유형

| 유형 | 설명 |
|------|------|
| 학생 | 앱 사용자 (출석, 스케줄, 급식, 멘토링 신청 등) |
| 학부모 | 자녀 정보 열람 및 급식·멘토링 신청 가능 |

## 핵심 요구사항

### [A] React Native 앱 전환 (공통)
- Expo 기반 iOS/Android 앱 쉘 구성
- 기존 Next.js 반응형 웹을 WebView로 임베드
- 앱 아이콘, 스플래시 스크린 설정
- App Store / Google Play 배포

### [B] 네이티브 푸시 알림
- FCM(Android) + APNs(iOS) 연동
- 공지사항 등록 → 자동 푸시 발송
- 채팅 메시지 수신 → 푸시 알림
- 알림 탭 → 앱 내 해당 화면 딥링크 이동

### [C] 자동 로그인 / 세션 관리
- Expo SecureStore 기반 토큰 영구 저장
- 앱 재실행 시 자동 로그인
- 비밀번호 재설정 (6자리 코드 방식)

### [D] 파일·사진 첨부
- 채팅: 이미지·파일 전송 (카메라/갤러리)
- 공지사항: 관리자 파일·이미지 첨부
- 파일 미리보기 / 다운로드

### [E] 급식 신청·결제 (학생·학부모)
- 센터별 중식/석식 메뉴 목록 조회
- 나이스페이먼츠 연동 결제
- 식사일 2일 전까지 취소·환불
- 신청 내역 조회

### [F] 급식 관리 (관리자 웹)
- 급식 상품 등록 (센터, 식사유형, 가격, 판매기간)
- 메뉴 입력
- 신청 현황 조회 및 관리
- 관리자 상시 취소

### [G] 기존 웹 기능 개선 (미완료 항목)
- **몰입도**: 지난 시간 추가 입력 / 핸드폰 제출 항목 / 학생 리포트 노출 / 과목별 학습시간 띠그래프
- **상벌점**: 30점 강제퇴원 알림·상계 / 분기별 초기화 / 100점 상품권 초기화 / 학부모 메뉴 상벌점 / 관리자 UX 개선
- **스케줄**: 일회성 자동 삭제 / 정기스케줄 기간 표기 / 관리자 화면 개선 / 결석 기능 추가
- **채팅**: 출석부 채팅방 연결 / 메시지 삭제 / 자주 쓰는 멘트
- **회원**: 미승인→입반 전환 / 탈퇴 오류 수정 / 지점 변경 / 학부모 정렬
- **오류 수정**: 과목 오입력 리셋 / 목표 학습시간 미반영

### [H] 멘토링/클리닉 신청 시스템 (추가 요청)

**학생·학부모 앱**
- 멘토링 캘린더 (신청 가능 슬롯 날짜별 표시)
- 슬롯 선택 (시간대·멘토·잔여 좌석)
- 신청자 정보 입력 및 신청 완료
- 신청 내역 조회 (신청중/확정/취소)
- 신청 취소 (취소 가능 시간 전)

**관리자 웹**
- 슬롯 등록 (멘토·날짜·시간대·과목·유형·정원)
- 주간 캘린더 뷰 (슬롯별 신청 현황)
- 신청 확정·거절 처리 및 알림 발송
- 신청 내역 목록 (날짜별·학생별 필터)
- 관리자 강제 취소

## 기술 스택

| 구분 | 기술 |
|------|------|
| 앱 쉘 | React Native (Expo) |
| 웹 임베드 | Expo WebView |
