# Step 2: 주요 개발 범위 — 루트스터디앱 통합 플랜

> 기존 8주 플랜 + 멘토링/클리닉 신청을 통합한 단일 개발 범위입니다.

---

### 1. React Native 앱 쉘 (학생·학부모 공통)

- Expo 기반 iOS/Android 앱 프로젝트 초기 구성
- 기존 Next.js 반응형 웹 임베드 (WebView 연동)
- 앱 아이콘 / 스플래시 스크린 설정
- 딥링크 처리 (알림 탭 → 앱 내 화면 이동)
- App Store / Google Play 빌드 및 배포

### 2. 네이티브 푸시 알림 시스템

- Expo Push Notification (FCM/APNs) 설정 및 토큰 관리
- 공지사항 신규 등록 시 자동 푸시 발송
- 채팅 메시지 수신 시 푸시 알림
- 서버 측 푸시 발송 API 구축
- 알림 수신 후 앱 내 해당 화면 딥링크 이동

### 3. 자동 로그인 / 세션 관리

- Expo SecureStore 기반 토큰 영구 저장
- 앱 재실행 시 자동 로그인 처리
- 비밀번호 재설정 (6자리 코드 방식)

### 4. 파일·사진 첨부 기능

- 채팅방: 이미지·파일 전송 (카메라/갤러리 접근)
- 공지사항: 관리자 파일·이미지 첨부 업로드
- 파일 미리보기 / 다운로드 처리

### 5. 급식 신청·결제 시스템 (학생·학부모)

- 급식 메뉴 목록 조회 (센터별 중식/석식)
- 급식 신청 및 결제 (나이스페이먼츠 연동)
- 신청 취소 및 환불 (식사일 2일 전까지)
- 신청 내역 조회

### 6. 급식 관리 (관리자 웹)

- 급식 상품 등록 (센터, 식사유형, 가격, 판매기간)
- 메뉴 입력 및 수정
- 신청 현황 조회 및 관리
- 관리자 상시 취소 처리

### 7. 멘토링/클리닉 신청 시스템 (학생·학부모)

- 멘토링 캘린더 화면 (신청 가능 슬롯 날짜별 표시)
- 슬롯 선택 (시간대·멘토·유형·잔여 좌석 확인)
- 신청자 정보 입력 및 신청 완료
- 신청 내역 조회 (신청중/확정/취소 상태별)
- 신청 취소 (취소 가능 시간 전)

### 8. 멘토링 스케줄 관리 (관리자 웹)

- 슬롯 등록 (멘토·날짜·시간대·과목·유형·정원)
- 주간 캘린더 뷰 (슬롯별 신청 현황 확인)
- 신청 확정·거절 처리 및 알림 발송
- 신청 내역 목록 (날짜별·학생별 필터)
- 관리자 강제 취소

### 9. 기존 웹 기능 개선 (미완료 항목 일괄)

- **몰입도**: 지난 시간 추가 입력 / 핸드폰 제출 항목 / 학생 리포트 노출 / 과목별 학습시간 띠그래프
- **상벌점**: 30점 강제퇴원 알림·상계 / 분기별 초기화 / 100점 상품권 초기화 / 학부모 메뉴 상벌점 / 관리자 UX
- **스케줄**: 일회성 자동 삭제 / 정기스케줄 기간 표기 / 관리자 화면 개선 / 결석 기능 추가
- **채팅**: 출석부 채팅방 연결 / 메시지 삭제 / 자주 쓰는 멘트
- **회원**: 미승인 입반 전환 / 탈퇴 오류 / 지점 변경 / 학부모 정렬
- **오류 수정**: 과목 오입력 리셋 / 목표 학습시간 미반영

### 10. 시스템 통합 및 배포

- 나이스페이먼츠 결제 API 연동
- Expo 앱 빌드 파이프라인 구성 (EAS Build)
- App Store / Google Play 심사 제출
- 서버 푸시 알림 인프라 구성
