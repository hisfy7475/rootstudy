# 0330 루트 스터디센터 통화 정리

## 요청 사항 요약

### 1. 몰입도 리포트 — 번호순 정렬 기능 추가
- **문제**: 학생 수가 많아 일괄 출력 시 번호가 꼬임
- **요청**: 번호순으로 정렬해서 출력할 수 있는 기능

### 2. 몰입도 리포트 — "향후 지도 계획" 란 출력 개선
- **문제**: "학부모 상담요약"은 내용 길이에 따라 페이지가 자동으로 늘어나는데, "향후 지도 계획"은 박스 크기가 고정되어 내용이 잘리고 스크롤해야 볼 수 있음
- **요청**: 학부모 상담요약처럼 내용 양에 따라 페이지가 늘어나도록 수정

### 3. 채팅 내용 복사 불가 문제
- **문제**: 학생 채팅 내용을 클릭해도 복사가 안 됨 (대표님 쪽에서는 복사가 된다고 했지만 클라이언트 쪽에서는 안 됨)
- **요청**: 채팅 내용 복사 가능하도록 수정

### 4. 몰입도 리포트에 채팅 관리 내용 연동 항목 추가
- **문제**: 채팅 관리에서 작성한 학생 관리 내용을 몰입도 리포트의 "향후 지도 계획"에 붙여넣고 싶은데, 복사도 안 되고 칸도 늘어나지 않아서 이중 작업 발생
- **요청 (우선)**: 몰입도 리포트에 채팅 관리 내용이 자동 반영되는 항목을 추가
- **요청 (차선)**: 최소한 복사 + 향후 지도 계획 칸 확장이라도 해결

## 답변 필요
