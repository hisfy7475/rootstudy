# 05. 스크린샷 / 아트워크 사양

> 모든 이미지는 **PNG (RGB, 알파 없음)** 권장. 텍스트 오버레이는 한국어 사용.

---

## A. 앱 아이콘

| 용도 | 사양 | 현재 자산 위치 |
|------|------|----------------|
| **iOS — App Store 등록용** | **1024 × 1024 PNG, 알파 없음, 둥근 모서리 적용 금지** (Apple이 자동 처리) | `studycafe-app/assets/icon.png` |
| iOS — 빌드 내부용 | Expo가 자동 생성 | (자동) |
| **Android — Play Store 등록용** | **512 × 512 PNG, 알파 가능** | `studycafe-app/assets/icon.png` (1024 → 512 다운스케일 필요) |
| Android — Adaptive Icon | foreground / background / monochrome 분리 | `studycafe-app/assets/android-icon-*.png` (이미 준비됨) |

→ ✅ 현재 `assets/icon.png` 1024×1024가 있으면 그대로 활용. 알파 채널 있는지 확인 필요.

---

## B. App Store 스크린샷 (필수)

### B-1. 6.7" (iPhone 16 Pro Max) — **필수**

- **사이즈**: `1290 × 2796` 또는 `1320 × 2868` (포트레이트)
- **개수**: 최소 3장, 최대 10장 권장
- **콘텐츠 구성안**:
  1. 학생 홈 — "오늘 X시간 X분 공부했어요" 통계 카드
  2. 출결 화면 — 입실/퇴실 타임라인
  3. 학습 통계 — 주간 그래프
  4. 채팅 화면 — 담임/학부모 대화 (이미지 첨부 노출)
  5. 학부모 자녀 모니터링 — 자녀 학습 현황
  6. 급식 신청 — 메뉴 + 신청 화면
  7. 멘토링 신청 — 일정 캘린더

### B-2. 6.5" (iPhone 11 Pro Max) — **iPhone 자동 스케일 옵션 사용 가능**

- 사이즈: `1242 × 2688`
- 6.7" 등록 시 Apple이 자동 변환 가능 (Apple 정책 변경됨, 별도 업로드 권장은 그대로)

### B-3. iPad — 본 앱 미지원 (`supportsTablet: false`)

→ 업로드 **불필요**. App Store에서 "iPhone only"로 노출됨.

### B-4. App Preview (동영상, 선택)

- 15~30초, MP4, 사이즈는 스크린샷과 동일
- 결제·자녀 모니터링 등 흐름 설명에 효과적

## C. Google Play 스크린샷 / 그래픽

### C-1. Phone Screenshots — **필수**

- **사이즈**: `1080 × 1920` (16:9 / 9:16) 또는 `1080 × 2400` (모던 비율) 권장
- **개수**: 최소 2장, 최대 8장
- **구성**: B-1 의 화면을 동일하게 사용 (Play도 같은 콘텐츠 가능)

### C-2. 7-inch Tablet / 10-inch Tablet — 선택

- 본 앱이 태블릿 최적화 안 함 → **업로드 안 해도 됨**
- 업로드 안 하면 "Designed for phones" 라벨이 붙음

### C-3. **Feature Graphic — 필수**

- **사이즈**: `1024 × 500 PNG/JPG` (정확히 이 크기)
- **콘텐츠**: 로고 + "학습관리·출결·소통을 한 번에" 슬로건
- 마케팅 이미지로 Google Play 상단 노출

### C-4. **High-res Icon** (Play 상점용)

- **사이즈**: `512 × 512 PNG`, 32비트(알파 OK), 1024×1024 ZIP 허용
- 위 A 항목과 동일

### C-5. Promo Video (선택)

- YouTube URL만 입력 (파일 업로드 X)

## E. 디자인 가이드 한 줄

- **상태바·노치 포함된 실제 디바이스 프레임**보다 **앱 화면 100% 캡처 + 위쪽에 마케팅 카피 1~2줄 오버레이**가 클릭률 높음.
- 색감은 앱 메인 컬러 `#4F46E5` (인디고) + 화이트 베이스로 톤 통일.
- 한국어 카피: "오늘 공부 4시간 30분", "자녀 출결 실시간 확인", "급식 신청도 한 번에" 등 짧고 명확하게.

## F. 빠른 캡처 방법 (참고)

| 디바이스 | 캡처 방법 |
|---------|----------|
| iPhone 시뮬레이터 (Xcode) | `Cmd + S` → 자동으로 정확한 사이즈 PNG |
| Android 에뮬레이터 | 에뮬레이터 컨트롤 패널 → 카메라 아이콘 |
| 실제 디바이스 | Xcode → Devices → "Take Screenshot" / Android Studio → Logcat 옆 카메라 |

> 시뮬레이터에서 **데모 계정으로 로그인 → 화면별 이동 → Cmd+S** 흐름이 가장 빠름.
